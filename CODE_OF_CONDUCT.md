# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful, assume good faith, and keep discussion focused on the work. Harassment,
personal attacks, and bad-faith behavior aren't welcome here.

Issues with conduct can be raised by opening an issue or contacting the maintainer via their
[GitHub profile](https://github.com/realdeal88). Reports will be handled discreetly.
